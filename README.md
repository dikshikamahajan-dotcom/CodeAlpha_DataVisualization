# CodeAlpha Task 3 - Data Visualization

## Project Title
Delivery Data Visualization using Python

## Objective
Visualize delivery data using charts and graphs to identify patterns and insights.

## Tools Used
- Python
- Pandas
- Matplotlib

## Visualizations
- Delivery Status Distribution
- Payment Method Distribution
- Delay Distribution
- Average Delay
- Top 10 Most Delayed Orders

## Dataset
Blinkit Delivery Dataset (Kaggle)

## Conclusion
The visualizations help understand delivery performance, delays, payment methods, and customer order trends.
